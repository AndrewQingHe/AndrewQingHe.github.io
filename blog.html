How to Write Posts

To create a new blog post:

  1. Create a Markdown (.md) file in the blog_articles directory
  2. Add metadata at the top (title, date, author, tags)
  3. Write your content using Markdown syntax
  4. Save the file - it will automatically appear in the blog
